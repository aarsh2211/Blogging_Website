 Blogging_Website
 
Created using the following technologies - 
ExpressJS
ReactJS
MongoDB
SocketIO

To run 

Step 1:
```
Clone the Repository
```

Step 2:
```
Navigate to backend and run npm install
```

Step 3:
```
Run npm start to deploy the REST APIS
```

Step 4:
```
Navigate to Frontend
```

Step 5:
```
Run npm install
```

Step 6:
```
Run npm start
```
